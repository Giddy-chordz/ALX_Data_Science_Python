🐍 Python Projects – Data Analysis

This directory contains Python-based projects and exercises completed during the ALX Data Science Program, with a focus on data analysis, cleaning, and visualization.

📌 Overview

Python is a powerful tool for data analysis.
This module focuses on using Python to:

Clean and prepare data
Perform exploratory data analysis (EDA)
Generate meaningful insights through visualization
🛠️ Tools & Libraries

The following tools and libraries are used across the projects:

Python 3.x

Pandas → Data manipulation and analysis

NumPy → Numerical computing

Matplotlib → Data visualization

Seaborn → Statistical visualization
